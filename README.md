# Título do Projeto

js-quiz

## Descrição
Aplicação web  -  quiz sobre javascript 
A aplicação consiste em um quiz interativo com o usuário, mostra o progresso de cada pergunta, temos o calculo final e o resultado com base no que foi respondido.
